# UnglamorousAI

Welcome. This blog is a plainspoken notebook about the boring AI work that actually changes businesses.

The goal here is to demystify agentic systems and show practical, repeatable ways developers and small-to-medium businesses use them in production. No keynote-stage hype. No AGI timeline theater. Just what works, what fails, and what is worth shipping.

Everything on the site is written by hand and grounded in day-to-day implementation experience.

## Quick Start

```sh
npm install
npm run dev
```

Local dev server starts at `http://localhost:4321`.

## Scripts

| Command | Action |
| --- | --- |
| `npm run dev` | Start local dev server |
| `npm run build` | Build production output to `dist/` |
| `npm run preview` | Preview production build locally |

## Stack

- Astro
- Tailwind CSS
- React islands where needed
- Markdown/MDX content-driven posts
